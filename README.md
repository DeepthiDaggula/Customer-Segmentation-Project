# Customer-Segmentation-Project
Customer Segmentation using Machine Learning with Python and K-Means Clustering.

## Overview
This project performs customer segmentation using the K-Means Clustering algorithm. Customers are grouped based on their annual income and spending score to identify different customer segments for targeted marketing.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization using Matplotlib and Seaborn
- Feature Scaling using StandardScaler
- K-Means Clustering
- Elbow Method to determine the optimal number of clusters
- Customer Segment Visualization
- Cluster Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Dataset
- Mall Customers Dataset
- Features:
  - CustomerID
  - Gender
  - Age
  - Annual Income (k$)
  - Spending Score (1–100)

## Project Workflow
1. Import Libraries
2. Load Dataset
3. Data Preprocessing
4. Exploratory Data Analysis (EDA)
5. Feature Selection
6. Feature Scaling
7. Elbow Method
8. K-Means Clustering
9. Cluster Visualization
10. Cluster Analysis
11. Save Results

## Results
- Identified different customer segments using K-Means clustering.
- Visualized customer groups based on annual income and spending score.
- Generated insights to support customer behavior analysis and targeted marketing.
